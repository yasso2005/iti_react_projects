html

HTML (HyperText Markup Language) is the standard language for creating and structuring web pages using tags and elements. It defines how content like text, images, and links appear in a browser. It is a markup language, not a programming language. This means it annotates text to define how it is structured and displayed by web browsers. It is a static language, meaning it does not inherently provide interactive features but can be combined with CSSfor styling and JavaScript for interactivity.


css

CSS is the language we use to style a Web page. CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files


js

JavaScript is a dynamic, high-level, and multi-paradigm programming language. Originally designed to add interactivity to web browsers, it has evolved into a ubiquitous, general-purpose language powering client-side and server-side applications (via Node.js), desktop software, and mobile development.
